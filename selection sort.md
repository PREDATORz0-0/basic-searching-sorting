# Selection  Sort

Selection sort psuedocode

```
# Write the psuedocode 
START
DEF SELECTION_SORT(ARR):
    N = LEN(ARR)
    FOR I IN RANGE(N):  
        MININDEX = I  
        FOR J IN RANGE(I+1, N): 
            IF ARR[J] < ARR[MININDEX]:  
                MININDEX = J  
        ARR[I], ARR[MININDEX] = ARR[MININDEX], ARR[I]
END
```
